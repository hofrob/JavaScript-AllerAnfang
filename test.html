--- layout: default title: asdf --- # Variablen ## Simple Variablen Variablen müssen deklariert werden: var kontaktName; Es können auch mehrere Variablen auf einmal deklariert werden: var kontaktName, kontaktNummer; Zur besseren Lesbarkeit sollte jede Variable in einer eigenen Zeile stehen. Die Deklaration mit `var` endet erst beim Strichpunkt (`;`). Bei der Deklaration können auch gleich Werte zugewiesen werden: var kontaktName = '', kontaktNummer = '+43 '; ## Komplexe Variablen Ein Array ist eine Liste von Werten: var filme = [ 'M', 'Es', 'Up', 'The Assassination of Jesse James by the Coward Robert Ford' ]; Arrays sind nicht sortiert und Elemente sind nicht verpflichtend vom selben Datentyp. Das heißt folgendes Array ist valide aber nicht empfehlenswert: var chaos = [ 4, 'Hello', 7.7 ];
